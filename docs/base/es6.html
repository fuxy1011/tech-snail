# 字符的 Unicode 表示法

至此, JavaScript 共有 6 种方法可以表示一个字符

'\z' === 'z' //true
'\172' === 'z' //true
'\x7A' === 'z' //true
'\u007A' === 'z' //true
'\u{7A}' === 'z' //true

# codePointAt()

正确处理 4 个字节储存的字符

var s = '𠮷a';

s.codePointAt(0);// 134071
s.codePointAt(1);// 57271
s.codePointAt(2);// 97
